= manual instructions: =
1. Upload `wp_to_klick-tipp_tagged-email-marketing.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. enter Klick-tipp credentials or HTML subscription code and license in Plugin
4. The license is checked calling https://saleswonder.biz/
5. The Klick-Tipp credentials are checked calling https://www.klick-tipp.com
